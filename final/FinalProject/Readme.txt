--- Made to Order Item Store ---

A store that allows you to custumize items with various factors like size, shape, color, filling, base and custom text.
Saves these items in a cart and calculates price based on the selected options.



Main Menu gives 5 options:
1. Create New Item
2. View Cart
3. Empty Cart
4. Complete Order
5. Quit

Create New Item will take you through a series of questions to create a custom item.
It then saves it to your cart.

View Cart will display your cart with all items and information about them, as well as the total price.

Empty Cart will empty your cart, allowing you to start again.

Complete Order will ask you for some additional details and output output a text file with a random order number and all the order information.
It will also empty your cart.

Quit will quit the program.
